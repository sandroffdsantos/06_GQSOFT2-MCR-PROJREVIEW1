# **TypeDoc - Code Documentation**

TypeDocs


TypeStrong/typedoc: Documentation generator for TypeScript projects. (github.com)

Installation:

```
npm install typedoc --save-dev
```

or specific version:

```
npx typedoc --version
```

Config:
 
```
{  
  "typedocOptions": {
    "entryPoints": ["./src"],
    "out": "TypeDoc"
  },
  ]
}
```

Run:

```
npx typedoc
```
